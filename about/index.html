About


Photo Reducer is a web application where in you can upload photos and compress them to a particular file size accurately. For example, if you have a photo of file size 5 MB, you can upload it to the website and compress it to say 3 MB or even less than 1 MB. The size of the photo is reduced by reducing image quality.

Photo Reducer is developed by Mobso.
